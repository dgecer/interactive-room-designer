# Lunara

## Interactive Perspective-Based Room Designer

Lunara is an interactive graphics application developed using PyQt5.
The project allows users to create and manipulate a perspective-based
room environment through real-time graphical interactions.

---

## Features

### Perspective System
- One-point perspective rendering
- Movable vanishing point
- Dynamic perspective grid system

### Room Generation
- Floor creation
- Ceiling generation
- Wall rendering
- Perspective-aware room structure

### Object System
- Perspective rectangle objects
- Perspective triangle objects
- Line drawing system

### Object Manipulation
- Object selection
- Object dragging
- Interactive resizing
- Rotation system
- Color changing
- Object deletion

### File Operations
- Scene saving using JSON
- Save dialog support
- Ctrl + S shortcut support

---

## Technologies Used

- Python
- PyQt5
- QPainter
- JSON

## Purpose of the Project

The goal of this project is to develop an interactive graphics
application that combines:

- Perspective rendering
- Real-time interaction
- Transformation systems
- GUI development
- Object-oriented design

The project was developed as part of a Computer Graphics course.

---

## Future Improvements

Planned future improvements include:

- Advanced 3D room systems
- Furniture libraries
- Texture support
- Multiple perspective modes
- Real-time lighting systems

In future versions, the project may also be integrated with
EEG-based emotion analysis systems.

The idea is to analyze a user's emotional state using brainwave
signals and automatically generate personalized room designs
according to the user's cognitive and emotional condition.

Possible future scenarios include:
- Relaxing room generation for stress reduction
- Personalized environments based on emotional state
- Adaptive lighting and color systems
- Neuroadaptive interior visualization

This could transform the project into a personalized
brain-computer interaction based interior design system.
